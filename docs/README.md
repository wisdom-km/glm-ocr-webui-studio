# Project Docs

This directory keeps long-lived project documentation out of the repository root.

## Structure

### `maintenance/`

Engineering-facing records for major repairs, architectural decisions, and maintenance history.

Current key document:

- `BUG_FIX_LOG.md`

### `troubleshooting/`

User-facing debugging guides and operator playbooks.

Current key document:

- `USER_PDF_DEBUG_PLAYBOOK.md`

## Related Log Folders

- `../logs/runtime/` stores stable application and backend runtime logs.
- `../logs/debug/` stores one-off smoke-test and debugging logs.
